A **Core Banking Web Application** developed using **Streamlit** and **MySQL** that allows users to register, log in, manage accounts, and perform basic banking operations.

## Features

- Passbook View (Credit, Debit & Balance)
- Deposit Money
- Withdraw Money
- Update Account Details

- MySQL Database Integration

---

## Technologies Used

- Python
- Streamlit
- MySQL
- Pandas
- streamlit-option-menu

---

## How TO Run

1. Install required packages: pip install -requirements.txt

2. Setup MySQL database using: database.sql

3. Run the app: streamlit run app.py
